Sobre o Projeto

Uma solução escolar para um problema global

Desenvolvido por estudantes como projeto de ciências, nosso sistema promete revolucionar a agricultura sustentável.

Irrigação Precisa

O sistema fornece água apenas quando necessário, evitando desperdício.

Tecnologia Acessível

Componentes de baixo custo tornam a solução viável para pequenos produtores.

Sustentabilidade

Redução no consumo de água e energia, contribuindo para o meio ambiente.

  • Válvula Solenoide - Controla o fluxo de água
